# OSCALic

Trying out OSCAL modeling with Python and Pydantic




---

## Building the Package

```
pip install --upgrade build
python3 -m build
```

## Install from Source

```
pip install git+https://github.com/Compton-NIST/OSCALic.git@develop#egg=oscalic
```

## Local Install

```
pip install -e .
```
